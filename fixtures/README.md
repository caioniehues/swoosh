# Swoosh fixture corpus

> Location note: DERISK §2 specifies `tests/fixtures/`; the corpus lives at top-level `fixtures/`
> instead, because `tests/` case-collides with SwiftPM's `Tests/` directory on case-insensitive
> macOS filesystems. Same role, collision-free path.

Each `*.json` file here is a **fixture**: a self-contained capture of one gesture or short
session that records exactly what the suppress/pass gate saw, plus the **correct** decision for
each event. The headless replayer (`Sources/SwooshFixtures/Replayer.swift`) feeds every fixture
back through the recognizer in CI — with no trackpad and no Accessibility — and fails if the
recognizer now decides differently than the recorded baseline. **A diff is a regression**
(DERISK §2–3). This is also the body of the macOS-beta canary (DERISK §4).

Fixtures are the lowest-bar way to contribute: no toolchain required, just a JSON file.

## Schema (v1)

```jsonc
{
  "schemaVersion": 1,
  "meta": { "name": "...", "description": "why this fixture exists", "recordedOnOS": "macOS 26.5", "capturedAt": "2026-05-31" },
  "samples": [
    {
      "t": 0.016,            // seconds from the start of the capture (time alignment)
      "contactCount": 2,     // Layer 2: active finger count
      "phase": 1,            // Layer 1: scroll phase — 1=Began, 2=Changed, 4=Ended, 8=Cancelled, 128=MayBegin
      "isContinuous": true,  // false = a discrete mouse wheel (never our gesture)
      "cursorX": 400, "cursorY": 14,             // cursor in AX global coords (top-left origin)
      "bandX": 0, "bandY": 0, "bandW": 800, "bandH": 28,  // Layer 3a: titlebar band under the cursor
      "decision": "suppress" // the recorded-correct outcome: "suppress" | "pass"
    }
  ]
}
```

- **Omit `bandX/Y/W/H`** entirely for a sample where the cursor is over no titlebar (the gate
  then passes — a missing band never suppresses).
- `decision` is the **intended** behavior you are asserting, not generated by the code. That is
  what lets the replay catch a recognizer that drifts away from it.
- Optional `rawContactFrame: [ints]` records the raw Layer-2 frame so a future struct-layout
  change reads as "raw present, decode mismatched" rather than a silent wrong answer.

## The suppress rule (what a correct `decision` follows)

Suppress **only if all** hold (SPEC §6.2), else pass:
1. `isContinuous == true`, and
2. `contactCount == 2`, and
3. `phase` is Began (1) or Changed (2) — never MayBegin (128), and
4. the cursor `(cursorX, cursorY)` lies inside the titlebar band.
